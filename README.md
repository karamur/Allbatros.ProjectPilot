# Allbatros Operations Hub

Allbatros Global icin ClickUp benzeri, yapay zeka destekli proje yonetimi uygulamasi.

## Ozellikler
- ClickUp baglanti ayarlari ve baglanti testi
- Kanban tabanli gorev panosu
- AI destekli gorev ozeti ve operasyon analizi
- Gorev detay ekrani ve risk gorunumu
- Blazor tabanli yonetim arayuzu

## Teknoloji
- .NET 10
- Blazor Web App

## Calistirma
```bash
dotnet restore
dotnet run --project Allbatros.OperationsHub.csproj
```

## Sonraki adimlar
1. Gercek `OpenAI` veya `Azure OpenAI` entegrasyonu eklemek
2. Gorev durum guncelleme ve yorum akisi eklemek
3. ClickUp webhook ve otomasyon modullerini tamamlamak
4. Dashboard metriklerini canli veriye baglamak
